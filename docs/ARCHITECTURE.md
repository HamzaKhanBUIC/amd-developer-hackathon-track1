# Architecture & Required Skills

## Required Skill Sets & Team Alignment

### 1. Python & FastAPI (Member 3 & AI)
- **Skills Needed:** `fastapi-pro`, `python-patterns`
- **Objective:** Build a highly concurrent, type-safe API using Pydantic. Ensure low-latency responses for the router.

### 2. Machine Learning & Embeddings (Lead & Member 3)
- **Skills Needed:** `scikit-learn`, `xgboost`, embedding space math.
- **Objective:** Train the XGBoost classifier on the Chatbot Arena dataset to accurately predict LLM success rates. Setup Cosine Similarity for the Semantic Router.

### 3. Next.js & Frontend (Member 1 & 4)
- **Skills Needed:** `nextjs-best-practices`, `stitch-design-taste`
- **Objective:** Implement the App Router, handle server-side vs client-side rendering properly, and craft a Vanilla CSS UI that rejects generic templates (AI slop) in favor of premium glassmorphism.

### 4. DevOps & Cloud (Lead)
- **Skills Needed:** Docker, AMD Developer Cloud, ROCm setup.
- **Objective:** Ensure the containers build cleanly and can be deployed to the required hackathon environment seamlessly.
